# Reporting Dashboard Design

## Core KPIs
- Leads by source (`lead_source`)
- Leads by domain (`website_domain`)
- Conversion rate (lead -> deal -> closed won)
- Pipeline value by pipeline and stage
- Closed revenue by lead type and domain

## Dashboard panels
1. Weekly lead volume by source
2. Pipeline stage distribution by pipeline
3. Monthly closed won revenue
4. Conversion funnel by lead type
5. Attribution table (`website_domain`, `source_campaign`)

## Attribution convention
- Primary: `website_domain`
- Secondary: `source_campaign`
- Tertiary: `lead_source`

## Review cadence
- Weekly: pipeline movement + conversion blockers
- Monthly: domain-level ROI and campaign quality
